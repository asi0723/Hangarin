# 📋 Hangarin — Task & To-Do Manager

A simple web application built with Django that helps users organize their daily tasks, manage priorities, add notes, and break down large goals into smaller subtasks.

---

## 🚀 Features

### 📊 Dashboard Overview
- Live summary cards showing:
  - Total number of tasks
  - Total completed tasks
  - Total pending tasks
  - Tasks created in the current year
- Recent tasks table with quick access

### ✅ Full CRUD Support
Complete Create, Read, Update, and Delete operations for:
- Tasks
- SubTasks
- Notes
- Categories
- Priorities

### 🔍 Search Functionality
Search across all list views by:
- Task title
- Category
- Description
- Content (Notes)

### 🔽 Filtering
Filter tasks by:
- Status (Pending, In Progress, Completed)
- Priority
- Category

### 🔀 Sorting
Sort tasks by:
- Title (A-Z / Z-A)
- Date created (Newest / Oldest)
- Due date (Earliest / Latest)
- Status

### 🔐 Authentication
- Email & password login/signup
- GitHub OAuth login
- Login required for all pages

### ⚙️ Admin Panel
Fully configured Django Admin with:
- Search functionality
- Filtering options
- Customized list displays
- Proper model management
